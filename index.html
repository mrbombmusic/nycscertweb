Programming in a High Level Language

These were two projects which I was very happy with. I enjoyed the groups I was in and was very pleased with how the projects turned out. Mancala was a simulation of the game Mancala. The concept we were learning about was arrays. We went with two arrays for each side. This made the basic implementation of the board a bit tricky but once that was set, it was much easier to implement the rules of the game.

Conway's Game of Life was something I had heard about in the world of creative coding but never quite understood what it was. This used the concepts of 2D arrays, but we also got to do some very basic animation in the console which was fun to watch.
